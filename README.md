# 🎮 シンプル数字選択ゲーム

1～6の数字から3つを順次選択するシンプルなゲームです。ExpoGoで視覚的に楽しめます。

## 🎯 ゲームのルール

1. **1桁目**: 1～6の数字から1つ選択
2. **2桁目**: 1桁目で選択した数字以外から1つ選択
3. **3桁目**: 1桁目と2桁目で選択した数字以外から1つ選択

## 🚀 セットアップ

### 必要な環境
- Node.js (v16以上)
- Expo CLI
- Expo Go アプリ（スマートフォン）

### インストール手順

1. **依存関係のインストール**
```bash
npm install
```

2. **Expo CLIのインストール（初回のみ）**
```bash
npm install -g @expo/cli
```

3. **アプリの起動**
```bash
npm start
```

4. **Expo Goで確認**
   - スマートフォンにExpo Goアプリをインストール
   - QRコードをスキャンしてアプリを開く

## 🎮 ゲームの特徴

### 視覚的表示
- **スロット形式**: 選択した数字がスロットマシンのように表示
- **アニメーション**: 数字選択時にスロットが回転
- **グラデーション**: 美しいグラデーション背景

### ゲーム機能
- **順次選択**: 1桁目→2桁目→3桁目の順で選択
- **重複防止**: 既に選択した数字は選択不可
- **AI提案**: スマートな数字提案機能
- **結果分析**: 選択した数字のパターン分析

### パターン分析
- **連続数字**: 1,2,3 や 4,5,6 など
- **奇数/偶数**: 奇数の個数による分類
- **合計値**: 数字の合計による分類

## 📱 使用方法

1. **ゲーム開始**: アプリを起動
2. **数字選択**: 1～6の数字ボタンをタップ
3. **順次選択**: 3つの数字を順番に選択
4. **結果確認**: 選択完了後に結果とパターンを表示
5. **再プレイ**: 「もう一度プレイ」ボタンでリセット

## 🎨 カスタマイズ

### 色の変更
`App.js`の`LinearGradient`の`colors`プロパティを変更:
```javascript
colors={['#667eea', '#764ba2']}  // 現在の色
colors={['#ff6b6b', '#feca57']}  // 例: 赤→黄
```

### アニメーション速度
`animateSlot`関数の`duration`を変更:
```javascript
duration: 100,  // 現在の速度（ミリ秒）
duration: 200,  // より遅く
```

## 🐛 トラブルシューティング

### よくある問題

1. **QRコードが表示されない**
   - `npm start`を実行
   - ターミナルに表示されるQRコードを確認

2. **Expo Goでアプリが開かない**
   - スマートフォンとPCが同じWi-Fiに接続されているか確認
   - Expo Goアプリが最新版か確認

3. **アニメーションが動かない**
   - デバイスの性能を確認
   - アプリを再起動
